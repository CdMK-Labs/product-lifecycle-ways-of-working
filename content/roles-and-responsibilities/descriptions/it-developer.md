## Purpose

The IT Developer is a technical expert specialising in areas such as software development, data engineering or SAP, responsible for delivering working software increments as well as providing monitoring and production support.

## Responsibilities

- Writing clean, testable code with unit and integration tests and clear documentation
- Maintaining code quality through code reviews, refactoring and active management of technical debt
- Implementing and maintaining CI/CD pipelines
- Ensuring system observability, performance and reliability
- Adhering to security, privacy and compliance standards
- Providing production support including incident triage and root-cause analysis

## Deliverables

- Working software increments
- Technical design documents
- Automated test suites
- Deployment scripts and manifests
- Monitoring dashboards and alerts
- Release notes

## Measured on

- Lead time for changes
- Code quality metrics
- Test coverage
- Throughput and predictability

## Reporting line

Reports to Head of Product, Product Architect or IT Delivery Lead.
